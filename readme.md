The app for execution of math expressions.

## Run locally

Install dependencies:

```sh
yarn install
```

Run in dev mode:

```sh
yarn dev
```

Run tests mode:

```sh
yarn test
```

Build:

```sh
yarn build
```

Update badges in readme.md:

```sh
yarn make-badges
```
